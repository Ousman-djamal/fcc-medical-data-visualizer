# Medical Data Visualizer

A data analysis and visualization project built with **Python**, **Pandas**, **Matplotlib**, and **Seaborn** as part of the **freeCodeCamp Data Analysis with Python** certification.

## 📌 Project Overview

This project analyzes a medical examination dataset and generates two different visualizations:

- **Categorical Plot (Cat Plot)** to compare health indicators based on cardiovascular disease.
- **Correlation Heatmap** to identify relationships between medical variables.

The project demonstrates common data analysis tasks such as:

- Data cleaning
- Feature engineering
- Data transformation with Pandas
- Correlation analysis
- Statistical visualization

## 📂 Dataset

The project uses the **Medical Examination Dataset** provided by freeCodeCamp.

The dataset contains information about patients, including:

- Age
- Height
- Weight
- Blood pressure
- Cholesterol
- Glucose level
- Smoking habits
- Alcohol consumption
- Physical activity
- Cardiovascular disease

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Features

- Calculate Body Mass Index (BMI)
- Create an **overweight** feature
- Normalize cholesterol and glucose values
- Generate a categorical plot
- Generate a correlation heatmap
- Remove invalid and outlier data before analysis

## 🚀 How to Run

Clone the repository:

```bash
git clone https://github.com/Ousman-djamal/fcc-medical-data-visualizer.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python main.py
```

## ✅ Tests

This project includes unit tests provided by freeCodeCamp.

Run the tests with:

python main.py

All tests should pass successfully.
